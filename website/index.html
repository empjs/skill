Why @nova/skill?

Unified Storage

All your skills live in ~/.nova-agent/skills/. Centralized management for distributed AI intelligence.

Instant Symlinking

Automatically detects and symlinks skills to all installed agents: Claude Code, Cursor, Windsurf, and more.

Dev Mode

Develop locally with --link. Changes to source files reflect immediately in your AI sessions.

How to Use

Terminal

1 Install a Skill

# Install from NPM (automatically detects all agents)
nova-skill install @nova/rn-skill
# Or install for a specific agent
nova-skill install @nova/rn-skill --agent claude
Terminal

2 Developer Mode

Link a local skill for development. Changes reflect immediately.

# Go to your skill package
cd packages/nova-rn-skill
# Link it
nova-skill install . --link
Terminal

3 List Installed Skills

nova-skill list
