# EMP Skill Platform - Web 平台功能指南

## 🌐 平台概述

EMP Skill Platform 是一个专注于 AI Agent 技能发现和社区互动的 Web 平台，提供以下核心功能：

- 🔍 **技能发现** - 浏览、搜索和推荐技能
- ❤️ **社区互动** - 点赞、收藏、评论技能
- 📊 **数据可视化** - 下载量、评价、排行榜
- 👥 **用户社区** - 开发者展示、用户交流
- 🚀 **零门槛体验** - 大部分功能无需登录

## 🎯 核心功能

### 1. 技能浏览（无需登录）

#### 首页展示
访问 https://skill.empjs.dev 查看：

- **热门技能** - 按下载量排序的技能排行榜
- **最新技能** - 最近更新的技能
- **分类浏览** - 前端、后端、DevOps、AI/ML 等分类
- **搜索功能** - 支持关键词搜索

#### 技能卡片信息
每个技能卡片显示：
- 技能名称和描述
- 开发者信息（GitHub 用户名）
- 下载量和点赞数
- 支持的 AI Agents
- 最后更新时间

### 2. 技能详情页

- 分类和标签
- 创建和更新时间

#### 安装指南
提供多种安装方式：

```bash

# GitHub 安装
```

### 3. 社区互动（需登录）

#### 用户认证
支持两种登录方式：

**Google 登录**
- 点击 "Google 登录" 按钮
- 授权 EMP Skill Platform 访问基本信息
- 自动创建账户

**GitHub 登录**
- 点击 "GitHub 登录" 按钮
- 授权 EMP Skill Platform 访问公开信息
- 自动创建账户

#### 点赞功能
- 点击 ❤️ 按钮为技能点赞
- 显示点赞数量和点赞用户头像
- hover 查看点赞用户名
- 可以取消点赞

#### 收藏功能
- 点击收藏按钮将技能加入收藏夹
- 访问 `/favorites` 查看所有收藏
- 按分类筛选收藏的技能
- 可以取消收藏

## 🔧 开发和部署

### 技术栈
```typescript
// 前端框架
React 19 + TanStack Start + TanStack Router

// 样式系统
Tailwind CSS v4 + Lucide Icons

// 部署平台
Cloudflare Pages + Cloudflare D1 + Cloudflare KV
```

### 本地开发
```bash
# 安装依赖
pnpm install

# 启动开发服务器
pnpm dev

# 构建生产版本
pnpm build
```

### 部署流程
```bash
# Cloudflare Pages 自动部署
git push origin main

# 或手动部署
npx wrangler pages deploy .output/public
```
